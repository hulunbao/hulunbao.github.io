几个提高效率的Linux命令别名

2019年7月14日
| Linux
| 阅读

在Linux环境下工作，会有几个经常用到的繁琐的命令。而且还需要重复这些繁琐的命令。

Linux下我们知道有别名命令 alias ,可以将这些繁琐且常用的命令自定义为我们容易记住的别名。

下面介绍几个比较实用的命令别名。

  • 压缩文件，tar命令在Linux环境下使用比较常见，但是tar的参数比较多，所以我们可以起个别名叫untar,这样我们下次解压时，直接 untar filename 即可。

    alias untar='tar -zxvf'

  • 我们下载一个很大的文件时，会遇到因为网络而中断的情况，需要重新下载。是不是有点头疼？我们可以用 wget 的 -c 选项支持断点下载，我们将他设置为

    alias wget='wget -c'

  • 下载一个文件之后，我们想要校验下它的 checksum 值，可以设置别名为

    alias sha='shasum -a 256'

  • linux 下 ping 命令会一直输出，没有多大意义。我们可以利用 -c 选项限制输出次数

    alias ping='ping -c 5

  • 我们想随时随地启动一个 web服务器，我们可以用这个别名

    alias www='python -m SimpleHTTPServer 8000'

  • 我们有时候想测试下网速，但是Linux没有自带的命令可用，我们可以用第三方工具speedtest-cli 。

    sudo pip3 install speedtest-cli
sudo pip3 install speedtest-cli --upgrade #更新到最新版本

    我们先使用 sppedtest-cli 命令来选择离我们最近的服务器，然后设置下别名

    alias speed='speedtest-cli --server 2406 --simple'

  • 获取公网IP

    alias ipe='curl ipinfo.io/ip'

  • 清屏，可以用 crtl + l 快捷键，也可以将命令 clear 定义的更短。

    alias c='clear'

这些命令你不一定都用得上，每个人工作内容不一样，可以选用自己能用上的。
